# Agentic30 Fable 5 MVP ICP

## 핵심 사용자

macOS를 쓰는 전업 1인 개발자. AI coding tool로 만들 수는 있지만 고객 수요를 아직 명확히 검증하지 못했다.

## 필수 적합 조건

- 실제 local project path가 있다.
- agentic coding tool을 쓴다.
- customer, problem, current surface, next validation action을 좁히고 싶다.
- 명시적 privacy control이 있는 local work-memory capture를 받아들인다.

## 핵심 pain

사용자는 소프트웨어를 만들 수 있다. 병목은 누구에게 무엇을 보여주고, 어떤 반응을 확인하며, 오늘 무엇을 해야 하는지 결정하는 일이다.

## 강한 신호

- 실제 project folder를 선택한다.
- customer/problem answer가 구체적이다.
- "unknown" 또는 "not enough evidence" 상태를 받아들인다.
- scan output을 믿기 전에 source quote를 원한다.
- 최근 작업 맥락을 local로 search하고 싶어 한다.

## Anti-ICP

- 주말 사이드 프로젝트 사용자.
- 팀 workflow 구매자.
- 이미 revenue optimization 단계의 SaaS.
- Windows/Linux-only 사용자.
- App이 outreach나 payment를 대신 실행하길 원하는 사용자.
- local capture 자체를 거부하는 사용자.
